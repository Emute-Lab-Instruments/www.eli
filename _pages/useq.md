Info about useq

## `ain2`

Returns the value of analogue input 2

Example: to echo analogue input 2 to analogue output 2
```
(a2 (ain2))
```

## `swm <index>`

Read the value of a momentary switch

| Parameter | Description | Range |
| --- | --- | --- |
| index | The index of the switch | 1 or 2 |
